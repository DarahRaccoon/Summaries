# Resúmenes de asignatura

En este repositorio se encuentran los resúmenes de cada asignatura de la licenciatura Ingeniería en Biotecnología impartida en la Universidad Politécnica de Sinaloa (UPSIN).

Se espera que sea de ayuda tanto para alumnos para mejorar el aprendizaje como para docentes para facilitarles la enseñanza de las materias.

## Mejora cooperativa

Cualquier persona puede apoyar en la mejora de estos resúmenes, tanto dejando comentarios de mejora como enviando cambios. Estos procesos se explican a continuación:

### Comentarios sobre aspectos a mejorar

De encontrarse errores en algún documento, estos se pueden marcar mediante como un Issue.

### Envios de mejoras

Se pueden enviar mejoras de los documentos mediante un Pull Request.
