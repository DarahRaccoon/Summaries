Es un ciclo vital de una célula y es una serie de etapas de desarrollo y crecimiento entre su nacimiento y reproducción.

\section{Fases}

\begin{enumerate}
    \item (M) Fase mitótica: Se separa y divide el ADN para formar dos células
    \item (I) Interfase: Crecimiento celular
    \item (G):
        \begin{itemize}
            \item G0: Control
            \item G1: Crecimiento físico y preparación
            \item S: Replicación del ADN
            \item G2: Más crecimiento y síntesis de proteínas y organelos
        \end{itemize}
\end{enumerate}

\section{Tipos}

\begin{itemize}
    \item Mitosis
        \begin{itemize}
            \item Células somáticas
	        \item Las "hijas" son idénticas a la "madre"
	            \begin{itemize}
	                \item Profase: Condensación de filamentos de cromatina para generar cromosomas. Desaparece además el nucléolo y aparece el huso mitótico.
		            \item Metafase: Alineación de los cromosomas por los centrosomas.
		            \item Anafase: Separación de las cromátidas.
		            \item Telofase: Citocinesis ("Cierre" de las células)
	            \end{itemize}
        \end{itemize}
    \item Meiosis
        \begin{itemize}
            \item Gametos
	        \item Dos divisiones celulares consecutivas
        \end{itemize}
\end{itemize}
