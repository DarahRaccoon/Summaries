Resúmenes para Ingeniería en Biotecnología

Material de estudio abierto, colaborativo y en constante mejora

Este proyecto busca recopilar lo aprendido durante las asignaturas de la carrera y compartir resúmenes de tales, esto con el objetivo de crear recursos accesibles para alumnos y docentes. Todo el contenido está disponible para su descarga gratuita y toda persona puede colaborar en hacer de estos recursos fuentes más sólidas y confiables de información.

💬 ¿Cómo puedes ayudar a mejorar estos resúmenes?

Tus comentarios, correcciones y sugerencias son bienvenidos con mucho aprecio, y existen varios medios por los cuales puedes aportar, elige el que te sea más cómodo.

Formulario

Puedes enviar correcciones, sugerencias o incluso archivos a través del siguiente formulario.

Formulario

PDF's

Abre cualquier PDF en Drive, selecciona el texto y añade comentarios sobre errores u opiniones de mejora.

Drive

e-Mail

También puedes enviarme opiniones, correciones o información que quieras que se añada a los resúmenes directamente a mi correo.

Enviar correo

GitHub

Si tienes una cuenta de GitHub y conocimiento sobre LaTeX, puedes reportar errores (Issues) o directamente proponer cambios en los archivos (Pull Requests).

Repo Issues

¿Necesitas ayuda con GitHub o LaTeX?

Si te interesa contribuir de manera más directa en la mejora de estos resúmenes pero no estás familiarizado con GitHub y/o LaTeX, escríbeme sin problema, te puedo enviar materiales de apoyo y/o dar alguna asesoría en el uso de estas herramientas, las cuales incluso son de utilidad en ámbitos más allá de este proyecto.
